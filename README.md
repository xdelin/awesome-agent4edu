## <a id='intelligent-tutoring'></a>Intelligent Tutoring

> AI tutors, educational assessment, and personalized learning assistants.

- **[claude-educational-ai-skills](https://github.com/ChatGPT3a01/claude-educational-ai-skills)**

  - **Description**: Python-based tool using Claude to analyze student test data (JSON) and generate personalized, motivating feedback reports (PDF).
  - **Use Case**: Teachers needing specific improvement suggestions for each student beyond just scores.

- **[ELA-Tutor-Agent](https://github.com/cozette-brown/ELA-Tutor-Agent-System-Prompting-Experiment)**
  - **Description**: An experimental System Prompt transforming Claude into a professional English Language Arts (ELA) tutor.

- **[quiz-master](https://github.com/tanmaypatil/quiz-master)**
  - **Description**: App generating general knowledge quizzes using GenAI and Claude Code.
  - **Use Case**: Quick classroom quizzes, self-testing, flashcard creation.

- **[apprenticemode](https://github.com/yavzius/apprenticemode)**
  - **Description**: A unique skill to detect user "Knowledge Gaps". Infers what you don't know from your Prompt.
  - **Use Case**: Metacognitive learning, checking for missing knowledge, self-assessing coding level.

- **[EduBase](https://github.com/EduBase/MCP)**
  - **Description**: Interact with EduBase, an e-learning platform with quizzing and exam management capabilities.
  - **Use Case**: Taking quizzes, managing courses.

- **[Canvas LMS](https://github.com/ahnopologetic/canvas-lms-mcp)**
  - **Description**: Connects to the Canvas Learning Management System to access course data, assignments, and announcements.
  - **Use Case**: Checking grades, managing assignments.

- **[Educational Tutor MCP](https://github.com/iamisaackn/AI-TutorMCP)**
  - **Description**: AI-powered educational toolkit built on the Model Context Protocol (MCP).
  - **Use Case**: Personalized tutoring, educational assistance.
