## üè´ Campus & Student Management

Connecting school systems, managing schedules and assignments.

### LMS Integration
Direct access to Learning Management Systems like Canvas, Brightspace, etc. within AI.

- **[makenotion/notion-mcp-server](https://github.com/makenotion/notion-mcp-server)** (Official)
  Official Notion MCP for managing study notes and task boards.

- **[akshsgaur/CMUCanvasMCPSErver](https://github.com/akshsgaur/CMUCanvasMCPSErver)**
  Canvas LMS integration allowing students to query courses, assignment deadlines, and todos via AI chat (designed for CMU but adaptable).

- **[pranav-vijayananth/brightspace-mcp-server](https://github.com/pranav-vijayananth/brightspace-mcp-server)**
  Connects to Brightspace account to retrieve course information.

- **[MisterCommand/mcp-canvas-lms](https://github.com/MisterCommand/mcp-canvas-lms)**
  Canvas LMS access tool written in TypeScript.

- **[markmusic27/stanford-mcp](https://github.com/markmusic27/stanford-mcp)**
  Stanford course catalog query tool, including course grading information.

### Student Affairs & Planning
Managing academic progress and campus life.

- **[tejpalvirk/student](https://github.com/tejpalvirk/student)**
  Knowledge graph-based student context management, linking courses, assignments, and exams.

- **[Nitishroy-7033/Student_MCP_Server](https://github.com/Nitishroy-7033/Student_MCP_Server)**
  Modular college management system interface covering fees, subjects, and exam structures.

- **[abhijeetgiram/mcp-student-leave-copilot-ab](https://github.com/abhijeetgiram/mcp-student-leave-copilot-ab)**
  Student leave application management tool integrated with GitHub Copilot.

## üìö Study Tools & Assistants

### Quiz & Exam Prep
Auto-generate quizzes to assist in exam preparation.

- **[cardea-mcp/ExamPrepAgent](https://github.com/cardea-mcp/ExamPrepAgent)**
  Intelligent exam preparation companion using LLM to help students prepare for various certification exams.

- **[RidiculousBuffal/quizAgent](https://github.com/RidiculousBuffal/quizAgent)**
  AI-enhanced quiz agent supporting SSO login.

- **[kkjzio/mcp-quiz-generator](https://github.com/kkjzio/mcp-quiz-generator)**
  Generates questions in Markdown format based on requirements, convertible to HTML and Word quiz files.

- **[sheikhcoders/interleaved-learning-mcp](https://github.com/sheikhcoders/interleaved-learning-mcp)**
  Comprehensive learning tool: includes study scheduling, quiz generation, flashcard shuffling, and learning tracking.

- **[PratyayRajak/todopomo-mcp](https://github.com/PratyayRajak/todopomo-mcp)**
  Study aid tool combining To-Do lists with the Pomodoro technique.

### Language Learning
Assisting foreign language acquisition.

- **[w41ch0ng/MandarinMCP](https://github.com/w41ch0ng/MandarinMCP)**
  Chinese (Mandarin) learning tool covering HSK levels 1-6 vocabulary, supporting progress tracking and Anki export.

- **[satish-kori-27/swedish-language-mcp-server](https://github.com/satish-kori-27/swedish-language-mcp-server)**
  Swedish language learning assistance.

- **[Jonathan-Racaud/Glossa](https://github.com/Jonathan-Racaud/Glossa)**
  A generic language learning MCP server.

### Math & Science
Deep learning tools for specific subjects.

- **[clouatre-labs/math-mcp-learning-server](https://github.com/clouatre-labs/math-mcp-learning-server)**
  Math teaching server supporting matrix operations and data visualization, with a persistent workspace.

- **[Augmented-Nature/GeneOntology-MCP-Server](https://github.com/Augmented-Nature/GeneOntology-MCP-Server)**
  Bioinformatics tool accessing Gene Ontology (GO) data for ontology analysis and enrichment studies.

- **[cyanheads/clinicaltrialsgov-mcp-server](https://github.com/cyanheads/clinicaltrialsgov-mcp-server)**
  Search and retrieve clinical trial data from ClinicalTrials.gov, suitable for medical research.

- **[ebrandler/mcportfolio](https://github.com/ebrandler/mcportfolio)**
  A powerful tool for finance students offering 9 professional tools for portfolio optimization and quantitative analysis.
