## <a id='academic-writing'></a>ðŸ“š Academic & Writing

> Academic research, paper writing, and creative content creation.

- **[Arxiv MCP Server](https://github.com/blazickjp/arxiv-mcp-server)**
  - **Description**: A specialized MCP server for searching, retrieving, and analyzing academic papers from the arXiv database.
  - **Use Case**: STEM researchers needing fast access to latest preprints directly within Claude.

- **[ZotLink](https://github.com/TonybotNi/ZotLink)**
  - **Description**: Production-ready MCP server for Zotero. Saves open preprints with rich metadata and smart PDF attachments.
  - **Use Case**: Academics managing large bibliographies wanting to automate "paper-to-library" workflow.

- **[Firecrawl MCP](https://github.com/firecrawl/firecrawl-mcp-server)**
  - **Description**: Advanced web scraping/crawling MCP turning websites into LLM-ready markdown. Handles dynamic content.
  - **Use Case**: Building custom datasets from the web for research or analysis.

- **[BrowserBase MCP](https://github.com/browserbase/mcp-server-browserbase)**
  - **Description**: Cloud-based headless browser for AI. Allows running browser automation scripts in a secure cloud environment.
  - **Use Case**: Automating complex web tasks that require a full browser context.

## <a id='coding-data'></a>ðŸ’» Coding & Data

> Computer science education, data analysis, visualization, and technical design.

- **[GitHub MCP Server](https://github.com/modelcontextprotocol/servers/tree/main/src/github)**
  - **Description**: The official GitHub integration for MCP. Manage issues, PRs, and analyze repository content directly.
  - **Use Case**: Learning version control workflows, automated code review.

- **[Playwright MCP](https://github.com/executeautomation/mcp-playwright)**
  - **Description**: Browser automation server enabling Claude to browse the web, test sites, and take screenshots.
  - **Use Case**: Learning automated testing, scraping data for projects.

- **[Desktop Commander](https://github.com/wonderwhy-er/DesktopCommanderMCP)**
  - **Description**: Deep control of your desktop OS. File management, terminal execution, and window control.
  - **Use Case**: Learning command line operations, automating local file tasks.

- **[Kubernetes MCP](https://github.com/fluxninja/mcp-kubernetes)**
  - **Description**: Interact with Kubernetes clusters. View logs, describe pods, and manage resources.
  - **Use Case**: Advanced DevOps learning, cluster monitoring.

- **[Sentry MCP](https://github.com/sentry-demos/mcp-server-sentry)**
  - **Description**: Retrieve Sentry issues and errors to debug applications faster with context.
  - **Use Case**: Learning production debugging and error tracking.

- **[Octocode MCP](https://github.com/bgauryy/octocode-mcp)**
  - **Description**: MCP server for semantic code research across repositories, allowing you to find real implementations.
  - **Use Case**: CS students learning how specific algorithms are implemented in open-source projects.

- **[MindsDB MCP](https://github.com/mindsdb/mindsdb)**
  - **Description**: Connects Claude to MindsDB's AI layer for databases. Train models and make predictions via SQL.
  - **Use Case**: Learning Applied Machine Learning with standard SQL syntax.

- **[SQLite Explorer](https://github.com/hannesrudolph/sqlite-explorer-fastmcp-mcp-server)**
  - **Description**: Safe, read-only access to SQLite databases. Explore tables and schema without risk.
  - **Use Case**: Beginners learning Database schemas and SQL queries.

- **[DBHub](https://github.com/bytebase/dbhub)**
  - **Description**: Database version control and collaboration tool. Like GitHub for your database schema.
  - **Use Case**: Learning database migrations and schema lifecycle management.

- **[Neon MCP](https://github.com/neondatabase/mcp-server-neon)**
  - **Description**: Serverless Postgres operations via MCP. Scale specific branches of data for testing.
  - **Use Case**: Modern cloud-native database development.

> PowerPoint generation, visual design, and presentation tools.

- **[Git MCP](https://github.com/idosal/git-mcp)**
  - **Description**: Tools to read, search, and manipulate local Git repositories. Visualize history and diffs.
  - **Use Case**: Understanding local git history, debugging complex merges.

- **[Docker MCP](https://github.com/ckreiling/mcp-server-docker)**
  - **Description**: Manage Docker containers, images, and volumes directly from Claude chat.
  - **Use Case**: Learning containerization concepts, debugging dev environments.

- **[Figma Context MCP](https://github.com/GLips/Figma-Context-MCP)**
  - **Description**: Extracts operational context from Figma designs to help Claude understand UI structure.
  - **Use Case**: Translating design mockups into code learning.

- **[pptx-mcp](https://github.com/samos123/pptx-mcp)**
  - **Description**: Simple MCP server to create slides using Python PPTX library.
  - **Use Case**: Quickly generating basic slide decks from text outlines.

- **[mcp-server-okppt](https://github.com/NeekChaw/mcp-server-okppt)**
  - **Description**: Generates high-quality PPTX slides from SVG images created by LLMs. Ensures vector quality.
  - **Use Case**: Creating visually designing slides with complex diagrams directly from Claude.

- **[Office-PowerPoint-MCP-Server](https://github.com/GongRzhe/Office-PowerPoint-MCP-Server)**
  - **Description**: A powerful MCP server for PowerPoint manipulation using python-pptx. Create, edit, and manipulate presentations.

## <a id='productivity-career'></a>ðŸ§  Productivity & Career

> Personal productivity, knowledge management, and career development.

- **[Slack MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/slack)**
  - **Description**: Interact with Slack channels and messages. Summarize threads or draft replies.
  - **Use Case**: Learning team communication management.

- **[Discord MCP](https://github.com/v-v-vishnu/discord-mcp-server)**
  - **Description**: Connects Claude to Discord servers. Read messages and manage communities.
  - **Use Case**: Managing study groups or student communities.

- **[Linear MCP](https://github.com/jerhadf/linear-mcp-server)**
  - **Description**: Manage tasks and issues in Linear. Create tickets from conversations.
  - **Use Case**: Students learning Agile/Scrum project management.

- **[Todoist MCP](https://github.com/actionsOnGoogle/todoist-mcp-server)**
  - **Description**: Manage your Todoist tasks directly. Add homework or project deadlines.
  - **Use Case**: Personal task tracking and deadline management.

- **[NotebookLM MCP](https://github.com/PleasePrompto/notebooklm-mcp)**
  - **Description**: Integrates Claude with Google's NotebookLM allowing your AI agent to research your own documentation with grounded answers.
  - **Use Case**: "Talking" to your lecture notes or textbooks to prepare for exams using your specific source material.

- **[Official Notion MCP Server](https://github.com/makenotion/notion-mcp-server)**
  - **Description**: Official MCP server from Notion. Enables reading pages, searching databases, and creating content directly from Claude.
  - **Use Case**: Organizing class schedules, updating research wikis, or tracking application progress.

- **[Anki MCP Server](https://github.com/nailuoGG/anki-mcp-server)**
  - **Description**: Connects Claude to Anki via AnkiConnect. Generate flashcards from conversation and automatically add them to decks.
  - **Use Case**: Turning lecture notes or vocab lists instantly into reviewable flashcards.

- **[PDF Reader MCP](https://github.com/SylphxAI/pdf-reader-mcp)**
  - **Description**: High-performance PDF processing server optimized for speed (10x faster) and accuracy in text extraction.
  - **Use Case**: Quickly summarizing long textbooks, extracting data tables, or converting assignments.

- **[linkedin-mcp](https://github.com/pegasusheavy/linkedin-mcp)**
  - **Description**: Model Context Protocol (MCP) server for managing LinkedIn profile, skills, and education background.
  - **Use Case**: Personal branding, auto-updating career history.
