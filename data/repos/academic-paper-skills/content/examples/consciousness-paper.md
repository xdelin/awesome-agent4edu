# Example: Planning a Consciousness Paper

This example demonstrates using `academic-paper-strategist` to plan a philosophy paper on consciousness and mortality.

## Initial Prompt

```
User: Plan a paper on how mortality generates consciousness. I want to argue
that death is not just a boundary condition but a generative constraint that
produces subjective experience.
```

## Phase 1: Platform Analysis

### Claude's Platform Recommendation

```
Based on your topic, I recommend the following platforms:

1. **PhilArchive** (Primary) ⭐
   - Perfect fit for philosophy of mind
   - Strong phenomenology community
   - No peer review barrier for preprints

2. **PhilSci-Archive** (Secondary)
   - Good for philosophy-science interface
   - If paper has empirical implications

3. **arXiv (q-bio.NC)** (Alternative)
   - Only if computational/neuroscience focus

Decision: PhilArchive selected
```

### Sample Paper Analysis

Claude analyzes 8-10 papers from PhilArchive, extracting:

```
STYLE GUIDE EXTRACTED:

Structure:
- Abstract: 150-250 words, standalone argument summary
- Sections: 5-7 main sections typical
- Length: 6,000-10,000 words for full papers

Argumentation:
- Clear thesis in introduction
- Engagement with major positions (IIT, GNW, HOT)
- Thought experiments welcome
- Technical terms defined on first use

Citations:
- Author-date preferred
- 25-40 references typical
- Mix of classic and recent (last 5 years)
```

## Phase 3: Outline Optimization

### Generated Outline

```
THE CONSTRAINT ALGORITHM: HOW MORTALITY GENERATES CONSCIOUSNESS

1. INTRODUCTION
   - The hard problem reframed: Why death matters
   - Thesis: Mortality is not boundary but generator
   - Roadmap

2. THE STANDARD VIEW
   - Death as limitation
   - Consciousness as information processing
   - The missing link

3. MORTALITY AS GENERATIVE CONSTRAINT
   - From thermodynamics to phenomenology
   - The 9-step algorithm
   - Why finite systems develop subjectivity

4. MECHANISMS
   - Time pressure → Value generation
   - Energy limits → Compression → Self
   - Boundary detection → Pain/pleasure

5. IMPLICATIONS FOR AI
   - The immortality problem

   - Summary of argument
   - Future research directions
```
