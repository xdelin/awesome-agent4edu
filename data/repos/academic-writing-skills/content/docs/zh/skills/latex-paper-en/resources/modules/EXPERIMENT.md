# 实验分析 (Experiment Analysis)

实验分析模块利用 LLM 的逻辑能力，能够将粗糙的数据点或未结构化的打点草稿，扩写为具备顶刊结构标准、连贯严谨的学术分析段落。

## 功能特性

- **叙事连贯性**：将分析提炼为带有导语结论的紧凑段落，强制使用 `\paragraph{}` 统领全段。
- **分析深度**：侧重与 SOTA 模型的基线对比、消融贡献度以及参数敏感性分析，杜绝流水账报数。
- **极端客观**：维护学术语言的严密与客观，杜绝任何诸如“碾压、绝杀、遥遥领先”等非学术词组表达。
- **排版合规**：严格恪守 IEEE 及顶级会议的排版范式，在段落内全面弃用 `\textbf{}` 及 `\begin{itemize}` 列表系统。

## 用法示例

将草拟的数据直接交给脚本：

```bash
python scripts/analyze_experiment.py "我们的方法到了 94.2%，比之前的 SOTA 模型B高了大约三个点。而且因为去掉了冗余模块，速度也变得更快。"
```

## 内核约束（供 LLM 遵守）

被触发时，底层模型会处于以下强约束之下：
1. 绝对忠实于所给数据，不对未知指标凭空捏造。
2. 以高概括度的 `\paragraph{核心结论概括}` 充当开头。
3. 使用一般现在时阐述既得优势结论。
4. 返回的内容将全部由可即插即用的纯净 LaTeX 语法构成。
