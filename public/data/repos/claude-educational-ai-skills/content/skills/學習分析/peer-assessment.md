# 同儕互評機制設計 (Peer Assessment Mechanism Designer)

## 技能描述

設計有效的同儕評量與回饋機制，培養學生批判思考、自我評量與建設性回饋能力。透過結構化的評分規準（Rubric）、評量者訓練與品質保證機制，提升互評的信效度與學習價值，同時減輕教師評量負擔。

## 適用情境

- **寫作互評**：作文、報告、論文初稿的同儕評閱與回饋
- **概念圖互評**：圖形組織作品的結構與內容品質評鑑
- **口頭報告評估**：簡報技巧、內容組織、表達清晰度的同儕評分
- **專案作品評量**：設計作品、程式作業、藝術創作的多維度評估
- **小組貢獻評量**：團隊協作中個人貢獻度的 360 度評鑑
- **線上討論品質**：論壇發言、討論參與的深度與品質評估
- **遊戲化互評**：結合積分、徽章系統的趣味化同儕評量
- **偏鄉應用**：紙本互評表單、實體回饋卡片的低技術實施方案

## 同儕互評流程架構

```
階段 1：準備階段
  ├─ 設計評分規準（Rubric）
  ├─ 準備標竿範例（不同等級作品）
  └─ 訓練評量者（口徑校正）
          ↓
階段 2：作品提交
  ├─ 設定提交規範（格式、期限）
  ├─ 去識別化處理（若採匿名）
  └─ 分配評量者（隨機/分層）
          ↓
階段 3：執行互評
  ├─ 使用 Rubric 評分
  ├─ 撰寫文字回饋
  └─ 教師監控品質
          ↓
階段 4：回饋整合
  ├─ 彙整多位評量者意見
  ├─ 學生閱讀回饋
  └─ 反思與修改作品
          ↓
階段 5：成績計算
  ├─ 作品成績（同儕分+教師分）
  ├─ 評量者成績（評量品質）
  └─ 學習反思成績
```

## 提示詞範本

### 📋 Rubric 評分規準設計

請為以下學習作品設計詳細的評分規準（Rubric）：

【作品類型】
[例如：國小五年級自然科「能源議題」概念圖]

【學習目標】
[例如：
1. 正確識別至少 5 種能源類型
2. 建立能源之間的階層關係
3. 標示能源優缺點與環境影響
4. 提出創新的交叉連結]

【評量目的】
- [X] 總結性評量（影響成績）
- [X] 形成性評量（提供回饋）
- [ ] 自我評量（反思學習）

【評量者】
- [X] 同儕互評
- [X] 教師評分
- [ ] 自我評量

請設計 Rubric，包含：

1. **評量向度設定**（4-6 個向度）

   請識別關鍵評量向度，例如：
   - 向度 1：概念完整性（是否涵蓋關鍵概念）
   - 向度 2：階層結構（上位-下位概念組織）
   - 向度 3：連結正確性（關係標示適當）
   - 向度 4：交叉連結（創造性整合）
   - 向度 5：視覺呈現（清晰易讀）

2. **等級量表設計**

   選擇適當等級數：
   - [ ] 3 等級（超越/符合/接近標準）
   - [X] 4 等級（優秀/良好/尚可/待改進）
   - [ ] 5 等級（卓越/優秀/良好/尚可/待改進）

3. **詳細 Rubric 內容**

   請為每個向度提供 4 個等級的描述：

   **向度 1：概念完整性**（權重 30%）

   | 等級 | 分數 | 描述（可觀察、可測量） |
   |------|------|----------------------|
   | 優秀 | 4 | 包含 8 個以上關鍵概念，涵蓋所有能源類型（太陽能、風能、水力、化石燃料等），並加入環境影響概念 |
   | 良好 | 3 | 包含 5-7 個關鍵概念，涵蓋多數能源類型，概念正確 |
   | 尚可 | 2 | 包含 3-4 個概念，部分能源類型遺漏，或有少數概念錯誤 |
   | 待改進 | 1 | 包含少於 3 個概念，或概念錯誤較多 |

   （為每個向度提供完整表格）

4. **學生友善版 Rubric**

   將專業用語轉換為學生易懂的語言：
   - 「階層結構」→「你的概念圖有沒有分類？大概念包含小概念？」
   - 「交叉連結」→「你有沒有找到不同分類之間的關係？」

5. **標竿範例說明**

   為每個等級提供作品範例或描述：
   - 優秀範例：「看看這個概念圖，它有...」
   - 良好範例：「這個概念圖做得不錯，如果能...會更好」

請提供：
1. 完整 Rubric 表格（含所有向度與等級）
2. 學生友善版 Rubric
3. 標竿範例說明（或範例作品描述）
4. 教師使用指引（如何使用此 Rubric 評分）

### 🎓 評量者訓練活動設計

設計訓練學生成為有效評量者的工作坊：

【訓練目標】
學生能夠：
1. 理解並正確應用評分規準
2. 撰寫具體、建設性、有幫助的回饋
3. 保持公平客觀的評量態度

【目標學習者】
[例如：國中八年級，30 人，即將進行作文互評]

【訓練時間】
[例如：1 節課 45 分鐘]

請設計訓練活動，包含：

1. **活動 1：回饋品質辨識**（15 分鐘）

   教師提供：
   - 1 件範例作品（如：一篇作文段落）
   - 5-6 則不同品質的回饋範例

   回饋範例類型：
   - **優秀回饋**：「第二段的比喻很生動（『時間像流水』），但第三段的論點可以加入具體例子支持，例如...」
   - **模糊回饋**：「寫得很好」「需要改進」（無具體指引）
   - **過於批評**：「這段寫得很差，完全看不懂」（負面語氣）
   - **過於寬鬆**：「全部都很棒！沒問題！」（無實質建議）

   學生任務：
   1. 閱讀所有回饋
   2. 小組討論：哪些回饋最有幫助？為什麼？
   3. 歸納「高品質回饋」的特徵

   教師總結：提出「高品質回饋檢核清單」
   - [ ] 具體指出位置（「在第二段...」）
   - [ ] 說明優點或問題
   - [ ] 連結評分規準
   - [ ] 提供改進建議
   - [ ] 語氣鼓勵與尊重

2. **活動 2：Rubric 應用練習**（15 分鐘）

   教師提供：
   - 2-3 件不同品質的範例作品
   - 評分規準 (Rubric)

   學生任務：
   1. 個人評分：使用 Rubric 為範例作品評分
   2. 小組討論：比較評分結果，討論差異
   3. 口徑校正：全班討論，教師說明標準答案與理由

   口徑校正重點：
   - 「優秀」與「良好」的界線在哪？
   - 如何判斷「概念完整性」？
   - 常見評分錯誤有哪些？

3. **活動 3：回饋寫作練習**（15 分鐘）

   提供回饋框架（TAG 法或肯定-建議-鼓勵法）：

   **TAG 回饋模板**：
   ```
   【T - Tell 我喜歡的部分】
   你的作品中，我覺得 ___ 很棒，因為 ___

   【A - Ask 我的疑問】
   我想問：___？ 或者我好奇 ___？

   【G - Give 我的建議】
   如果能 ___，你的作品會更好，例如 ___
   ```

   學生任務：
   1. 選擇一件範例作品
   2. 使用 TAG 模板撰寫回饋
   3. 兩人配對互相檢視回饋品質
   4. 全班分享優秀回饋範例

4. **總結與承諾**（5 分鐘）

   - 回顧高品質回饋的要素
   - 學生簽署「評量者承諾書」：
     - 我承諾公平評量
     - 我承諾給予建設性回饋
     - 我承諾尊重他人作品

請提供：
1. 完整訓練活動教案（含時間分配）
2. 回饋範例集（好與不好的對比）
3. 學生活動單（含 TAG 回饋模板）
4. 評量者承諾書範本

### 🔄 同儕互評流程管理

設計完整的同儕互評實施流程與管理機制：

【課程資訊】
- 科目：[例如：國文科]
- 作業類型：[例如：議論文寫作（800 字）]
- 班級人數：[例如：32 人]
- 互評時程：[例如：2 週（提交→互評→修改）]

【互評設定】
- 匿名評量：[X] 是  [ ] 否
- 每件作品評量者數：[例如：3 人]
- 每人評量件數：[例如：3 件]
- 成績佔比：作品 70%（同儕評分 40% + 教師評分 30%），評量品質 30%

請設計互評流程，包含：

1. **階段 1：作品提交**（第 1-3 天）

   提交規範：
   - 格式：Word 檔案，12pt 新細明體，行距 1.5 倍
   - 檔名：學號_姓名_議論文.docx
   - 截止時間：第 3 天晚上 11:59
   - 提交平台：[例如：Google Classroom / 學校學習平台 / 紙本]

   去識別化處理（若匿名）：
   - 教師移除檔案中的姓名資訊
   - 重新編號：A01, A02, A03... （使用代碼）

2. **階段 2：評量者分配**（第 4 天）

   分配策略：
   - [X] 隨機分配（避免朋友互評偏誤）
   - [ ] 能力分層（高/中/低能力互評）
   - [ ] 互惠評量（A 評 B，B 評 A）

   分配結果通知：
   - 每位學生收到 3 件作品代碼（如：A05, A12, A23）
   - 下載作品檔案或領取紙本
   - 收到評分規準與回饋模板

3. **階段 3：執行互評**（第 5-10 天）

   評量任務：
   - 使用 Rubric 為每個向度評分
   - 撰寫文字回饋（每件作品至少 100 字）
   - 提供具體改進建議

   教師監控：
   - 每天檢視提交進度
   - 抽檢評量品質（10-20% 作品）
   - 發現問題及時介入（如：敷衍評分、不當回饋）

   學生支援：
   - 設立答疑時段
   - 線上討論區（釐清評分標準）

4. **階段 4：回饋彙整與閱讀**（第 11-12 天）

   教師處理：
   - 彙整每件作品的 3 份評量
   - 計算平均分數
   - 標記極端評分（與平均差距過大）
   - 進行教師複核（抽檢或全數）

   學生收到：
   - 自己作品的 3 份同儕回饋
   - 平均分數與各向度得分
   - 教師補充意見（若有）

   反思任務：
   - 閱讀所有回饋
   - 填寫反思單：「哪些回饋最有幫助？為什麼？」
   - 規劃修改方向

5. **階段 5：修改與再提交**（第 13-14 天）

   修改任務：
   - 根據回饋修改作品
   - 撰寫「修改說明」（我根據回饋做了哪些修改？）
   - 提交最終版本

   成績計算：
   - 初稿成績（同儕評分平均）× 40%
   - 最終版成績（教師評分）× 30%
   - 評量品質成績 × 20%
   - 反思與修改 × 10%
   = 總成績

6. **品質保證機制**

   評量品質檢核：
   - 評分過於寬鬆或嚴格（與平均差距 > 1 分）→ 教師面談
   - 文字回饋過短（< 50 字）→ 提醒補充
   - 不當回饋內容（攻擊性、離題）→ 移除並重新分配
   - 優質回饋（被作者評為最有幫助）→ 加分獎勵

   學生申訴機制：
   - 認為評分不公→ 提出申訴→ 教師複核→ 調整分數
   - 收到不當回饋→ 通報教師→ 移除回饋→ 教師介入處理

請提供：
1. 完整流程時程表（甘特圖或時間軸）
2. 教師檢核清單（每階段要做什麼）
3. 學生任務清單（每階段繳交什麼）
4. 評分計算 Excel 範本

### ⚖️ 互評倫理與公平性設計

設計同儕互評的倫理規範與爭議處理機制：

【倫理關注點】
- 隱私保密（學生作品是否公開？）
- 評分公平（避免友誼/敵意偏誤）
- 心理安全（學生能否承受批評？）
- 學術誠信（避免串通作弊）

請設計倫理規範與處理機制：

1. **隱私保密規範**

   規則設計：
   - 互評作品僅評量者與教師可見（不公開全班）
   - 匿名評量優先（雙盲：作者與評量者均匿名）
   - 禁止截圖或分享他人作品
   - 違反規定的處理方式

2. **公平性保證策略**

   潛在偏誤與對策：

   | 偏誤類型 | 現象 | 對策 |
   |---------|------|------|
   | 友誼偏誤 | 給朋友高分 | 匿名評量、多評量者平均 |
   | 敵意偏誤 | 故意給低分 | 教師監控、極端值處理 |
   | 寬嚴偏誤 | 整體過鬆或過嚴 | 標準化訓練、口徑校正 |
   | 光環效應 | 所有向度給相同分數 | 提醒分向度獨立評分 |
   | 能力差異 | 低成就學生評量品質較低 | 加強訓練、教師複核 |

3. **心理安全支持**

   建立支持性文化：
   - 強調「回饋是幫助而非批評」
   - 教師示範接受回饋的態度
   - 使用「肯定-建議-鼓勵」結構
   - 提供情緒支持管道（輔導老師）

   弱勢學生保護：
   - 特殊需求學生的調整（如：可選擇退出排行榜）
   - 低成就學生的鼓勵（強調進步而非絕對分數）
   - 避免公開完整排名（只表揚前段，不揭露後段）

4. **學術誠信規範**

   禁止行為：
   - 串通作弊（互給高分）
   - 敷衍評量（隨意給分、複製貼上回饋）
   - 惡意評量（故意給低分、人身攻擊）

   偵測機制：
   - 統計分析：偵測異常互評模式（總是互相給高分）
   - 教師抽檢：檢視回饋品質
   - 同儕檢舉：提供匿名通報管道

   處理方式：
   - 初犯：警告、重新評量
   - 再犯：該次評量成績不計分
   - 嚴重違規：依校規處理

5. **爭議處理流程**

   **情境 1：學生認為評分不公**
   - Step 1：學生提出申訴（3 天內，需說明理由）
   - Step 2：教師檢視作品與回饋
   - Step 3：對照 Rubric 判斷
   - Step 4：決定維持或調整分數
   - Step 5：向學生說明理由

   **情境 2：收到不當回饋**
   - Step 1：學生通報教師
   - Step 2：教師審查回饋內容
   - Step 3：移除不當回饋
   - Step 4：與評量者面談教育
   - Step 5：重新分配評量者（若需要）

6. **學生須知文件**

   設計「同儕互評學生手冊」，包含：
   - 互評的目的與價值（為什麼要做互評？）
   - 評量流程說明（時程表）
   - 權利與責任（我可以做什麼？不可以做什麼？）
   - 倫理規範（隱私、公平、尊重）
   - 申訴管道（遇到問題找誰？）
   - FAQ 常見問題

請提供：
1. 倫理規範文件（可張貼在教室或平台）
2. 爭議處理流程圖
3. 學生互評須知手冊（Word 文件格式）
4. 家長說明信（若學生為國小/國中）

## 互評品質指標

| 指標 | 良好互評 | 不佳互評 |
|------|---------|---------|
| **評分一致性** | 與其他評量者差距 < 0.5 分 | 差距 > 1 分 |
| **回饋長度** | 100-200 字，充分說明 | < 50 字，過於簡短 |
| **回饋具體性** | 指出具體位置與內容 | 模糊籠統（「很好」「改進」） |
| **建設性** | 提供改進建議與方法 | 只批評，無建議 |
| **連結標準** | 參照 Rubric 說明 | 未連結評分規準 |
| **語氣** | 尊重、鼓勵、支持 | 負面、批判、攻擊 |

## 最佳實踐

### ✅ 實施建議
- 從低風險開始（形成性評量→總結性評量）
- 充分訓練後才正式實施（至少 1 節課訓練）
- 多評量者以提升信度（建議 2-3 位）
- 同儕評分不宜佔總成績過高（20-40%）

### ✅ 偏鄉友善實施
- 紙本互評表單（無須線上平台）
- 實體回饋卡片（手寫回饋）
- 簡化流程（減少複雜步驟）
- 善用小班優勢（教師可更密切監控）

### ✅ 品質保證
- 定期檢視評量品質（每次都抽檢）
- 收集學生回饋改進流程（每學期檢討）
- 分析信效度資料（進階應用）
