# 生成式 AI 教育應用 (Generative AI in Education)

## 技能描述
深入探討生成式 AI（如 Claude、GPT、Gemini）在教育領域的各種應用模式，從備課、教學、評量到學生自學。

## 教師端應用

### 1. 備課助手

#### 提示詞範本：課程大綱生成
```
請擔任資深[科目]教師，幫我設計一學期的課程大綱：

【課程資訊】
科目：[科目名稱]
年級：[年級]
每週節數：[節數]
學期週數：[週數]

【課綱依據】
核心素養：[素養代碼]
學習表現：[表現代碼]
學習內容：[內容代碼]

【學生背景】
程度描述：[描述]
先備知識：[列舉]

請設計包含：
1. 各週教學主題與目標
2. 重要概念的邏輯順序
3. 評量時間點安排
4. 跨領域連結機會
5. 差異化教學切入點
```

#### 提示詞範本：單元教材編寫
```
請幫我編寫以下單元的教材內容：

【單元資訊】
科目：[科目]
單元名稱：[名稱]
教學節數：[節數]
學習目標：
1. [目標1]
2. [目標2]
3. [目標3]

【教材需求】
- 引起動機素材
- 核心概念說明（適合[年級]學生閱讀）
- 生活化範例 3-5 個
- 常見迷思概念與澄清
- 重點整理

【風格要求】
- 語言程度：適合[年級]
- 語氣：[親切/正式/活潑]
- 特殊需求：[如：融入在地元素]
```

### 2. 評量設計

#### 提示詞範本：多元題型生成
```
請為以下主題設計多元評量題目：

【主題資訊】
科目：[科目]
主題：[主題名稱]
認知層次分布：
- 記憶理解：[%]
- 應用分析：[%]
- 評鑑創造：[%]

【題型需求】
1. 選擇題：[題數]
2. 是非題：[題數]
3. 配合題：[題數]
4. 填空題：[題數]
5. 簡答題：[題數]
6. 申論題：[題數]

【難度分布】
- 基礎：[%]
- 中等：[%]
- 挑戰：[%]

請提供：
- 完整題目
- 正確答案
- 評分標準（申論題）
- 試題分析（雙向細目表）
```

### 3. 個別化回饋

#### 提示詞範本：作業回饋生成
```
請針對以下學生作業提供個別化回饋：

【作業資訊】
科目：[科目]
作業主題：[主題]
評量規準：
1. [規準1]：[滿分]
2. [規準2]：[滿分]
3. [規準3]：[滿分]

【學生作答內容】
[貼上學生作業內容]

【學生背景】
學習程度：[高/中/低]
性格特質：[描述]
過去表現：[描述]

請提供：
1. 各規準評分與說明
2. 具體優點（至少2點）
3. 待改進處與建議（具體可行）
4. 鼓勵性結語
5. 延伸學習建議

回饋語氣：[鼓勵/中性/嚴謹]
```

## 學生端應用

### 1. AI 學習導師

#### 提示詞範本：概念學習
```
我是[年級]學生，正在學習[科目]的[主題]。

我的疑問是：
[學生的具體問題]

我目前的理解是：
[學生描述自己的想法]

請幫助我：
1. 不要直接給答案，用蘇格拉底式提問引導我思考
2. 如果我的理解有誤，溫和地指出並解釋
3. 用我能理解的例子說明
4. 給我一個練習題檢驗我的理解
```

### 2. 寫作夥伴

#### 提示詞範本：寫作輔助
```
我要寫一篇[文體]：

【題目】[題目或主題]

【寫作目的】[說明目的]

【目前進度】
[貼上目前的草稿]

請幫我：
- [ ] 檢查文章結構是否完整
- [ ] 指出論點不清楚的地方
- [ ] 建議可以增加的論據
- [ ] 檢查語句是否通順
- [ ] 提供改善建議（不要直接改寫）

注意：請用提問方式引導我自己修改，而不是直接給我修改後的版本。
```

### 3. 練習夥伴

#### 提示詞範本：互動練習
```
請當我的[科目]練習夥伴：

【練習主題】[主題]
【我的程度】[描述]
【練習目標】[目標]

規則：
1. 一次只給一題
2. 等我回答後再給反饋
3. 如果我答錯，給提示而非答案
4. 答對後提供變化題鞏固
5. 記錄我的正確率

開始吧！
```

## 最佳實踐

### 提示詞工程技巧

```
有效提示詞的要素：
┌─────────────────────────────────────┐
│  1. 角色設定 (Role)                  │
│     「請擔任資深國中數學教師...」     │
├─────────────────────────────────────┤
│  2. 任務說明 (Task)                  │
│     「幫我設計一份學習單...」        │
├─────────────────────────────────────┤
│  3. 情境脈絡 (Context)               │
│     「學生程度較弱，需要鷹架...」    │
├─────────────────────────────────────┤
│  4. 格式要求 (Format)                │
│     「請以表格呈現，包含...」        │
├─────────────────────────────────────┤
│  5. 範例提供 (Example)               │
│     「類似這種風格：...」            │
├─────────────────────────────────────┤
│  6. 限制條件 (Constraints)           │
│     「避免使用超過500字...」         │
└─────────────────────────────────────┘
```

### 品質檢核

```
AI 生成內容檢核清單：
□ 事實正確性：是否有錯誤資訊？
□ 適齡性：語言和內容適合學生年齡？
□ 文化敏感度：是否有偏見或不當內容？
□ 教學適切性：符合教學目標？
□ 原創性：不過度依賴 AI 原文？
□ 版權：是否涉及版權問題？
```

## 各 AI 模型比較

### 教育應用特性比較

| 特性 | Claude | GPT-4 | Gemini |
|------|--------|-------|--------|
| 長文處理 | ★★★★★ | ★★★★ | ★★★★ |
| 邏輯推理 | ★★★★★ | ★★★★★ | ★★★★ |
| 創意發想 | ★★★★ | ★★★★★ | ★★★★ |
| 中文能力 | ★★★★ | ★★★★ | ★★★ |
| 即時搜尋 | ✗ | ★★★★ | ★★★★★ |
| 多模態 | ★★★★ | ★★★★★ | ★★★★★ |
| 安全性 | ★★★★★ | ★★★★ | ★★★★ |
| 教育特化 | ★★★★ | ★★★★ | ★★★ |

### 場景選擇建議

| 教育場景 | 首選 | 備選 | 原因 |
|---------|------|------|------|
| 教案撰寫 | Claude | GPT-4 | 長文邏輯佳 |
| 創意活動設計 | GPT-4 | Claude | 創意多元 |
| 時事素材搜集 | Gemini | GPT-4 | 即時搜尋 |
| 數學推導 | Claude | GPT-4 | 步驟嚴謹 |
| 語言學習 | GPT-4 | Claude | 語言多元 |
| 圖像教材 | GPT-4 (DALL-E) | Gemini | 整合方便 |
