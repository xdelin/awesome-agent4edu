# 自適應學習設計 (Adaptive Learning Design)

## 技能描述
協助教師與課程設計者建構自適應學習系統，根據學生的學習表現、風格與需求，動態調整學習內容與路徑。

## 適用情境
- 設計線上學習平台的自適應機制
- 規劃差異化教學的學習路徑
- 建構個人化學習推薦系統
- 偏鄉教育資源有限環境下的適性化設計

## 核心概念

### 學習路徑模型
```
┌─────────────┐
│  診斷評估   │ ← 前測/學習風格測驗
└──────┬──────┘
       ↓
┌─────────────┐
│  能力分級   │ ← 初級/中級/進階
└──────┬──────┘
       ↓
┌─────────────┐
│  內容推薦   │ ← 依程度推送教材
└──────┬──────┘
       ↓
┌─────────────┐
│  即時回饋   │ ← 答題正確率/學習時間
└──────┬──────┘
       ↓
┌─────────────┐
│  路徑調整   │ ← 動態調整難度與內容
└─────────────┘
```

### 自適應參數設計

| 參數類型 | 說明 | 資料來源 |
|---------|------|--------|
| 認知程度 | 學生目前的知識水平 | 測驗成績、答題表現 |
| 學習速度 | 完成學習單元的速度 | 學習時間紀錄 |
| 學習風格 | 視覺/聽覺/動手操作 | 學習風格量表 |
| 錯誤模式 | 常見的錯誤類型 | 答題分析 |
| 參與度 | 學習投入程度 | 登入頻率、完成率 |

## 提示詞範本

### 設計自適應學習單元
```
請幫我設計一個自適應學習單元：

主題：[學習主題]
學科：[科目名稱]
目標對象：[年級/程度]
預計學習時間：[時間]

需求：
1. 設計 3 個難度等級的學習內容
2. 定義各等級的進入條件與晉升標準
3. 設計形成性評量問題（每等級 3-5 題）
4. 規劃學習路徑的分支邏輯
5. 設計即時回饋機制

特殊需求：[如：適合偏鄉低頻寬環境]
```

### 分析學生學習數據
```
請分析以下學生學習數據，並提供自適應建議：

學生資料：
- 前測成績：[分數]
- 學習時間：[時長]
- 答題正確率：[百分比]
- 錯誤題型分布：[類型列表]

請提供：
1. 學生目前能力診斷
2. 建議的學習路徑
3. 需要補強的概念
4. 推薦的學習資源類型
```

## 實作考量

### 偏鄉環境適配
- 支援離線學習模式
- 低頻寬友善的內容格式
- 多裝置相容（手機/平板/舊電腦）
- 本土語言/方言介面選項

### 教師端支援
- 學生學習進度儀表板
- 班級整體表現分析
- 預警系統（學習落後提醒）
- 教學調整建議
