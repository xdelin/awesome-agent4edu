# 數學科技教育 (Technology in Mathematics Education)

## 技能描述
探討如何運用科技工具提升數學教學效果，包括動態幾何軟體、代數計算系統、數學模擬平台等，協助學生理解抽象概念並培養數學思維。

## 數學科技工具分類

### 動態幾何軟體 (Dynamic Geometry Software)

| 工具 | 特色 | 適用主題 | 費用 |
|------|------|---------|------|
| **GeoGebra** | 免費、功能全面、跨平台 | 代數、幾何、統計、微積分 | 免費 |
| **Desmos** | 介面友善、活動分享 | 函數圖形、統計 | 免費 |
| **Cabri** | 歐洲主流、穩定 | 幾何探索 | 付費 |
| **Geometer's Sketchpad** | 美國主流、資源多 | 幾何證明 | 付費 |

### 代數計算系統 (Computer Algebra System)

| 工具 | 特色 | 適用場景 |
|------|------|---------|
| **Wolfram Alpha** | 強大計算、步驟顯示 | 解題驗證、探索 |
| **Symbolab** | 免費、步驟詳細 | 學生自學 |
| **Photomath** | 拍照解題 | 作業輔助 |
| **Microsoft Math** | 整合 Office | 作業批改 |

### 統計與數據工具

## 教學模式設計

### 探究式學習模式
```
┌─────────────────────────────────────────┐
│           科技融入探究學習               │
├─────────────────────────────────────────┤
│  1. 情境引入                             │
│     ↓ (科技呈現真實問題情境)            │
│  2. 提出猜想                             │
│     ↓ (學生觀察科技呈現的模式)          │
│  3. 操作探究                             │
│     ↓ (學生使用工具動手驗證)            │
│  4. 發現規律                             │
│     ↓ (科技輔助歸納數學性質)            │
│  5. 形式化                               │
│     ↓ (從具體到符號表徵)                │
│  6. 應用延伸                             │
│     (解決更複雜的問題)                   │
└─────────────────────────────────────────┘
```

### 翻轉教室模式
```
課前（非同步）
├── 觀看教學影片（概念講解）
├── 完成預習題（線上即時回饋）
└── 記錄疑問

課中（同步）
├── 教師回應疑問（快速講解）
├── 小組探究活動（科技工具輔助）
├── 問題解決任務（協作學習）
└── 即時評量回饋

課後（非同步）
├── 完成練習題（自適應平台）
├── 延伸探究（選做）
└── 反思筆記
```

## 常見迷思概念與科技介入

### 函數迷思與科技介入
| 迷思 | 科技介入策略 |
|------|-------------|
| 函數只能是公式 | 展示多重表徵（表格、圖形、公式） |
| 圖形就是函數本身 | 動態改變參數觀察 |
| 斜率是固定的數 | 切線動畫視覺化 |

### 幾何迷思與科技介入
| 迷思 | 科技介入策略 |
|------|-------------|
| 只認得標準位置圖形 | 動態旋轉、變形 |
| 證明只靠視覺直覺 | 拖曳驗證 vs 反例呈現 |
| 測量等於證明 | 比較測量與演繹推理 |
