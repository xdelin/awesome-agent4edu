## Line Editing: Common Patterns Addressed

### 1. Filter Words (Most Common Issue)
Removed or restructured filter words that create distance:

**Common filters removed:**
- "she felt" --> Direct description
- "she could feel" --> Direct sensation
- "she saw" --> Direct observation
- "seemed to" --> Definite action

**Examples:**
- "the iron seemed to shimmer" --> "the iron shimmered"
- "She could feel memories around her" --> "Memories shimmered around her"
- "Kira sensed him" --> "She sensed him"

### 2. Redundancy (Second Most Common)
Trimmed repeated or unnecessary information:

- "Just fatigue. She'd been up too early" --> "She'd been up too early"
- "She should have felt grief. She should have felt rage" --> "Grief should have come. Rage."

### 3. Weak Constructions
Strengthened verb choices and sentence structures:

- "hit harder than they should have" --> "hit like a blow"
- "thinking" + filter --> Direct thought

### 4. Rhythm Improvements
Adjusted sentence flow for better pacing:

- Shortened some constructions for cleaner flow
- Removed unnecessary connective tissue
- Varied sentence lengths for impact
