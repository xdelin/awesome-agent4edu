# Contributor Covenant Code of Conduct

Our project follows the Contributor Covenant (version 2.1). We expect all contributors to
be respectful, inclusive, and kind. Unacceptable behavior will not be tolerated.

Key points:
- Be respectful and patient.
- Assume good intent and be constructive.
- If you see harassment or problematic behavior, report it to the maintainers.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Contact: Please open an issue labeled `code-of-conduct` to contact the maintainers privately.
